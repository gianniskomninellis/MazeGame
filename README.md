🎮 Maze Escape
Introduction

Maze Escape is a simple maze-based game developed in Python using the Pygame library. The player controls a character and must navigate through a series of obstacles and walls to reach the finish area as quickly as possible.

The game features collision detection, a completion timer, and dynamic obstacles that make each attempt more challenging and engaging.

Features
Player movement using the W, A, S, and D keys.
Real-time collision detection with walls and obstacles.
Dynamic obstacle that changes size over time.
Completion timer that records the player's finishing time.
Smooth gameplay running at 60 FPS.
Simple and lightweight design.
Beginner-friendly project for learning Pygame and game development concepts.
Requirements

To run this project, you will need:

Python 3.8 or later
Pygame
Install Pygame
pip install pygame
How to Run
Clone this repository:
git clone https://github.com/your-username/maze-escape.git
Navigate to the project folder:
cd maze-escape
Run the game:
python maze_escape.py
Controls
Key	Action
W	Move Up
A	Move Left
S	Move Down
D	Move Right
Objective

Reach the yellow finish area while avoiding collisions with walls and obstacles. Once you reach the goal, your completion time is displayed and the game resets for another attempt.
